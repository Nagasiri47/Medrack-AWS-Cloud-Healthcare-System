# MedRack – AWS Cloud Healthcare System ☁️🏥

## 📌 Overview

**MedRack** is a cloud-based healthcare management system designed to securely store, manage, and access patient medical records using **Amazon Web Services (AWS)**. The system aims to improve healthcare data management by providing scalable infrastructure, secure storage, and easy access to patient information.

Cloud-based healthcare systems help hospitals and healthcare providers efficiently manage medical data, improve accessibility, and ensure secure handling of sensitive patient information. ([Akkodis][1])

This project demonstrates how cloud technologies can be integrated into healthcare applications to create a **reliable, scalable, and secure medical record management platform**.

---

# 🚀 Features

* 🧑‍⚕️ **Patient Record Management** – Store and retrieve patient medical records securely.
* ☁️ **Cloud-Based Architecture** – Hosted on AWS for scalability and reliability.
* 🔐 **Secure Data Storage** – Ensures safe handling of sensitive healthcare data.
* 📊 **Centralized Medical Information** – Easy access for healthcare professionals.
* 🏥 **Healthcare System Simulation** – Demonstrates how hospitals can use cloud systems for medical data management.

---

# 🏗️ System Architecture

The MedRack system is built using AWS cloud infrastructure to ensure scalability and high availability.

Typical architecture components may include:

* **AWS EC2** – Application hosting
* **AWS S3** – Secure storage for medical records
* **AWS RDS / DynamoDB** – Database for patient data
* **AWS IAM** – Secure access management
* **AWS CloudWatch** – Monitoring and logging

Cloud platforms like AWS allow healthcare systems to collect, store, and process medical data efficiently while providing scalable infrastructure for healthcare applications. ([Kien's Website][2])

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Nagasiri47/Medrack-AWS-Cloud-Healthcare-System.git
cd Medrack-AWS-Cloud-Healthcare-System
```

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

or

```bash
npm install
```

(depending on the backend stack)

---

## 3️⃣ Configure AWS

1. Create an **AWS Account**
2. Configure **IAM credentials**
3. Setup required AWS services:

   * EC2
   * S3
   * RDS

---

## 4️⃣ Run the Application

Example:

```bash
python app.py
```

or

```bash
npm start
```
